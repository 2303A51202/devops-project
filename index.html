Background

Four paradigms of
road scene understanding

Offline HD maps require manual annotation, are costly, and can't be updated in real time. SRSU is the first online framework to jointly extract both fine-grained map boundaries and lane topology.

01 / OFFLINE
Offline HD Map Construction
Uses LiDAR, IMU, GPS + manual annotation to build globally consistent HD semantic maps. Highly accurate but inflexible — cannot adapt to new roads or changing environments.
Labor-intensive · Not scalable
02 / ONLINE
Online Road Network Estimation
Parameterized curve theory extracts BEV lane centerlines with topology from single-view images. Limited to open-shaped elements — cannot accurately describe fine-grained map boundaries.
No map boundaries · Single-view only
03 / ONLINE
Online Road Map Detection
HD map construction theory extracts BEV road maps with fine-grained boundaries (road boundaries, lane dividers, pedestrian crossings). No topological information about lane connectivity.
No lane topology
OURS
04 / SRSU
Unified Hierarchical Map Construction
Jointly extracts fine-grained map boundaries and lane topology from surround-view images in a single online framework. No HD maps needed — complete structured road scene understanding.
Maps + Topology · Surround-view · Online
Architecture

How SRSU works

01
Map Encoder
ResNet-50 backbone extracts multi-view CNN features from 6 surround cameras. GKT lifts 2D image features to the BEV top-down plane, generating shared BEV features.
02
Hierarchical Map Decoder
Two sets of learnable hierarchical query embeddings — one for road map elements, one for lane centerlines. Each query combines point-level + instance-level information via cascading transformer layers with MHSA + Deformable Attention.
03
Multi-Task Learning
Two branches: Road Map Detection (classification + point regression) and Road Network Estimation (centerline detection + MLP-based adjacency matrix → lane topology graph).
04
Hierarchical Map Output
Road map: ordered point sets W=(U,Γ). Road network: directed graph G=(V,E) with adjacency probability matrix P∈ℝᴺˣᴺ encoding lane topology.

Empirical Modules

Three modules that
close the gap

Multi-task learning typically degrades vs single-task. These three modules are specifically designed to eliminate that degradation.

AT
Auxiliary Task Prediction
Two CNN decoders predict distance transform maps DRoad and DLane in rasterized BEV space. Supervises encoder training and supplies spatial priors downstream.
mAP gain+1.9%
APcenterline+2.1%
C-F (topology)+1.8%
D
Multi-Modal Distillation
Projects decoder reference points onto distance maps, encodes position via sinusoidal encoding + local direction via MLP, fuses spatial+semantic into enriched feature vectors.
mAP (with AT)+3.7%
APcenterline+3.5%
RequiresAT module
I
Higher-Order Interaction
Fully-connected global graph over all M+N feature vectors. Attentional GNN propagates via L layers of MLP + MHSA message passing — learning cross-task association.
mAP gain+2.2%
APcenterline+1.9%
C-F (topology)+2.0%
